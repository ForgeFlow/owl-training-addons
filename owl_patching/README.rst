Training for OWL Patching.

In this module we patch the Form view Controller (method patching) to
add some new behaviors in certain relevant methods (including the
display of notifications and dialogs). Moreover, we also patch the Email
widget (static properties patching) to add some new props to be taken
into account when rendering the widget.
