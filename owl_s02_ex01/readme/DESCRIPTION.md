OWL Training Session 02 - Exercise 01

In this exercise, we create a sales dashboard in Odoo containing the following
information:

- **Number of sales orders** in the selected month.
- **Number of new customers** in the selected month. New customers are defined
  as those whose first sale order was ordered in the selected month.
- **Table of sales by month** for the last 12 months, starting from the
  selected month. The table contains the following columns:
  - Date (month and year).
  - Number of sales orders.

By default, the selected month is the current month. The user can change the
selected month and year and the information in the dashboard will be updated
accordingly.
