OWL Training Session 03 - Exercise 01

In this exercise, we create a new hash generator field or widget in Odoo that
holds a char field.

This widget shows a new button on the right side of the input if it is empty.
Once we press it, a random value (string) is added to the field. We have also
added the possibility to set an option for different sizes of the random
generated value.

Moreover, if the field is readonly, the button should not be shown, and the
text of the field should not be editable either.
